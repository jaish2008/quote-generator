# ✨ QuoteLab — Advanced Quote Generator

QuoteLab is a **modern, interactive quote generator web application** built using **HTML, CSS, and JavaScript**.  
It delivers inspirational quotes with advanced UI effects, offline support, themes, animations, and personalization features.

---

## 🚀 Live Features

- 🎯 Random quote generator  
- 🎭 Mood-based quotes (Happy, Sad, Motivation, Love, Stress Relief)  
- 🌈 Multiple animated background themes  
- 🌙 Light / Dark mode toggle  
- 💾 Save favourite quotes (LocalStorage)  
- 📋 Copy & 📤 Share quotes (WhatsApp supported)  
- 🖼️ Download quote as an image (PNG)  
- 🎉 Confetti animation  
- 🔁 Auto-play quotes with interval control  
- 🎵 Optional background ambient music  
- 🧠 Daily quote (saved per day)  
- 🌌 Animated particle background  
- ⚡ Offline-ready fallback quotes  

---

## 🛠️ Technologies Used

- **HTML5** — Structure & accessibility  
- **CSS3** — Glassmorphism UI, animations, responsive layout  
- **JavaScript (ES6)** — Logic, API handling, LocalStorage  
- **Canvas API** — Particle animation & image export  
- **Web Audio API** — Sound effects & ambient music  
- **Font Awesome** — Icons

## 🌐 Quote Source

- **Offline Quotes** (default & fallback)
- **Live API**: `https://api.quotable.io/random`

> If the API is unavailable, QuoteLab continues working offline.

## ▶️ How to Run the Project

### Option 1: Open Locally
1. Download or clone the repository
2. Open `index.html` in any modern browser

### Option 2: Using Live Server (Recommended)
1. Open project in VS Code
2. Right-click `index.html`
3. Select **Open with Live Server**
